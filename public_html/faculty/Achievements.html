Project: Performance Improvement of Online Signature Recognition using SaaS Architecture The purpose of this project is to design a Cloud Architecture for online signature recognition, which will make the SRS system more Scalable, Pluggable and Faster. Moreover to add more accuracy and consistency in the verification process of the online signature recognition system, a classifier is also designed.
